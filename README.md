# Polycom Brute Force
A Simple Python Program that uses *requests* and *multithreading* to brute force **Polycom** Servers.
## Requirements
Language Used = Python3<br />
Modules/Packages used:
* requests
* warnings
* base64
* datetime
* urllib
* optparse
* colorama
* mutliprocessing
* time
<!-- -->
Install the dependencies:
```bash
pip install -r requirements.txt
```
